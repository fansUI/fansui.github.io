# 网络连接模式

设置位置：工具栏->编辑->虚拟网络编辑器

# 桥接模式：

  • 与真机处于同一 LAN 中，在宿主机中会生成一块虚拟网卡（vmnet0）

    虚拟机使用桥接模式相当于在局域网中新增一台电脑，虚拟机和物理机的地位一致，
    缺点：如果宿主机开的虚拟机太多，可能会导致 LAN 中 IP 地址冲突，因为局域网中 IP 地址数有限制，

# 仅主机模式：

在宿主机中会生成一块虚拟网卡（vmnet1），这块网卡相当于一块交换机，
所有设置了仅主机模式的虚拟机在同一局域网中，
虚拟机只能与宿主机和其他的设置了仅主机模式的虚拟机通信，不能与外网通信，
缺点：不能连接互联网，

# NAT 模式：

  • 网络地址转换模式，在宿主机中会生成一块虚拟网卡（vmnet8）

    可以访问互联网（外网），借助宿主机网卡与外网通信
