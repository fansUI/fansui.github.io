# eval 加密

原理：把源代码转为字符串形式，再把字符串当成一个函数的返回值，交给 eval 执行

eval 加密还原 (opens new window)

// 加密之前
function add(a, b) {
  return a + b;
}

add(1, 2);

// 转换过程
// 1.代码转字符串，这个字符串现在是一个整体，也可以把字符串进行拆分、编码等操作
let str = `
    function add(a,b){
        return a+b
    }
    add(1,2)
`;
// 2.构建函数，函数中可以增加花指令（代码填充）等冗余代码
function hahahaha() {
  // return str
  return `
    function add(a,b){
        return a+b
    }
    add(1,2)
`;
}
// 3.执行hahahaha函数
// eval(hahahaha())
eval(
  !(function hahahaha(a, b, c, d, e, f, g) {
    // return str
    return `
    function add(a,b){
        return a+b
    }
    add(1,2)
`;
  })()
);

# 内存爆破

  • 原理?

    使用 for 循环（死循环）使内存溢出
    setCookie：如果代码被格式化则开启死循环 getCookie：正则检测代码是否被格式化

  • 什么时候使用?

    检测代码被格式化，如果代码被格式化则开启爆破
    但是如果专业逆向者代码下载下来，去除此逻辑并借助 fiddler 依然可以运行
