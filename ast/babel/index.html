# babel 是什么？

Babel 是一个 JavaScript 编译器。
Babel 是一个通用的多用途 JavaScript 编译器，通过 Babel 你可以使用（并创建）下一代的 JavaScript。
Babel 是一个工具链，主要用于将 ECMAScript 2015+ 版本的代码转换为向后兼容的 JavaScript 语法，以便能够运行在当前和旧版本的浏览器或其他环境中。

# 为什么会有 babel？解决什么问题？

javascript 发展迅速，针对 es2015（es6）的新语法，除了谷歌火狐等新版本支持之外，不是所有的浏览器都支持 es6 语法，因此需要针对不支持 es6 语法的浏览器使用 Polyfill 的方式在目标环境中添加缺失的特性，或者把 es6 代码转换成 es5 代码，这样就可以解决浏览器不能识别 es6 语法的局面了，使用 babel 还可以转换 es7、es8 等更新的语法特性
